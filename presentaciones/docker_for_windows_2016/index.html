Microsoft Server 2016 features for docker

Windows Containers are offered with two container base images:

  1. Windows Server Core: Windows server apps (IIS, SQL Server, Full .NET Framework)
  2. Nano Server: New minimal Windows version (IIS, .NET Core)

Windows Container Types

  1. Windows Server Containers
  2. Hyper-V Containers

Windows Containers benefits for Docker

  1. Eliminate conflicts of different versions of IIS/.NET to coexist on a single system with container isolation.
  2. Complimentary with Hyper-V virtualization
  3. Take advantage of new base images like Windows Server Core and Nano Server
  4. Consistent Docker user experience use the same commands as Docker for Linux environments
  5. Added isolation properties with Hyper V containers selected at runtime
